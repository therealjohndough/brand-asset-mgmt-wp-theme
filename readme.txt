A lightweight, compliant starter theme for cannabis brands.

== Installation ==
1. Zip the `casestudylabs-cannabisbrandhub` folder and upload under Appearance → Themes → Add New.
2. Activate the theme.
3. (Optional) Set "Front page displays" to a static page using Front Page template.

== Configuration ==
Appearance → Customize → Cannabis Compliance → Market (state) to change the disclaimer.

== Notes ==
- Minimal CSS for speed. Extend in a child theme or by adding styles to `assets/css/theme.css`.
- State disclaimers live in `inc/compliance.php`.
